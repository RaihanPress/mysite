Do You Want to Add these two buttons to Your eClass?

So Please Follow these Steps.


What These Buttons do?

These two buttons are for skipping audio.

The first button will back 10 seconds from the audio.

And The Second button will forward 10 seconds from the audio.


How to Install These Buttons?

Click the Copy Button

Now on the Top Right ↗ Click on the ⋮ three dots

Click on the ☆ Star Button in Chrome

Click on Add to Bookmarks in Opera Mini

In Chrome, Click the Edit Button on Bottom.

Under the URL/Address, remove faizanahmad.tk/code and paste the code.

Under the Name, Type Any Letter.

Code is now Set Up!

How to Use These Buttons?

Every time you open Any Audio in eClass,

Click on the URL Bar.

Type the Letter you have Written as the Name

Your letter will be shown with a blue link starting with javascript:

Click on it

You Are all done!

Important:
You will have to re-type the name and re-select it every time you open any audio.
